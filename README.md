### Instalação

```bash
  # Clonar o repositório
  $ git clone

  # Entrar no diretório
  $ cd <pasta-do-projeto>

  # Instalar as dependências
  $ yarn ou npm install

  # Iniciar o projeto
  $ yarn dev ou npm run dev
```
